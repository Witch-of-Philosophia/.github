# 🪐 Witch of Philosophia  
### 위치 오브 필로소피아

《위치 오브 필로소피아》는 광활한 우주와 수많은 은하에서 벌어지는 사건을 해결하는  
**턴제 전략 RPG** 프로젝트입니다.

각 마녀는 고유한 개념과 철학을 지니고 있으며,  
플레이어는 **관찰자이자 기록자**로서 마녀들의 선택과 그로 인해 변화하는 세계를 지켜봅니다.  

한 번의 판단은 관계와 세계관의 질서를 바꾸고,  
그 파문은 서서히 축적되어 하나의 결말을 만들어냅니다.  

> 거대한 우주 속에서 아주 작은 의미가 어떻게 역사를 움직이는가?  
> 이 질문에 답하기 위해 우리는 오늘도 하나의 사건을 기록합니다.
